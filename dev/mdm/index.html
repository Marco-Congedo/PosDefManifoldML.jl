    Minimum Distance to Mean

    mdm.jl

    This unit implemets the Riemannian MDM (Minimum Distance to Mean) classifier for the manifold of positive definite matrices.

    Besides the MDM type declaration and the declaration of some constructors for it, this unit also include the following functions, which typically you will not need to access directly:

    (1)

    MDM machine learning models are incapsulated in this mutable structure. MDM models have two fields: .metric and .means.

    The field metric, of type Metric, is to be specified by the user. It is the metric that will be adopted to compute the class means.

    The field means is an â„Vector holding the class means, i.e., one mean for each class. This field is not to be specified by the user, instead, the means are computed when the MDM model is fit using the fit! function and are accessible only thereafter.

    (2)

    Constructor creating and fitting an MDM model with training data ğTr, an â„Vector type, and labels yTr, an IntVector type. The class means are computed according to the chosen metric, of type Metric. See here for details on the metrics. Supported metrics are listed in the section about creating an MDM model.

    Optional keyword arguments w and âœ“w are passed to the fit! function and have the same meaning therein.

    Examples:

    # (1)
# generate some data
ğTr, ğTe, yTr, yTe = gen2ClassData(10, 30, 40, 60, 80)

# create a model

# fit the model with training data

# (2) equivalently and faster:
ğTr, ğTe, yTr, yTe=gen2ClassData(10, 30, 40, 60, 80)

    Given a metric of type Metric, an â„Vector of Hermitian matrices ğ and an optional non-negative real weights vector w, return the (weighted) mean of the matrices in ğ. This is used to fit MDM models.

    This function calls the appropriate mean functions of package PostDefManifold, depending on the chosen metric, and check that, if the mean is found by an iterative algorithm, then the iterative algorithm converges.

    See method (3) of the mean function for the meaning of the optional keyword arguments w, âœ“w and â©, to which they are passed.

    The returned mean is flagged by Julia as an Hermitian matrix (see LinearAlgebra).

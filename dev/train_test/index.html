function predict(model  :: MLmodel,
                 ğTe    :: â„Vector,
                 what   :: Symbol=:labels)

Given a ML model model trained (fitted) on $z$ classes and a testing set of $k$ positive definite matrices ğTe of type â„Vector,

if what is :labels or :l (default), return the predicted class labels for each matrix in ğTe as an IntVector;

if what is :probabilities or :p, return the predicted probabilities for each matrix in ğTe to belong to a all classes, as a $k$-vector of $z$ vectors holding reals in $(0, 1)$ (probabilities).

Only the MDM model is supported for the moment being.

MDM model

For this model, the predicted class of an unlabeled matrix is the class whose mean is the closest to the matrix (minimum distance to mean).

The probabilities instead are obtained passing to a softmax function the distances of each unlabeled matrix to all class means.

See: notation & nomenclature, the â„Vector type.

See also: fit!, CVscore.
